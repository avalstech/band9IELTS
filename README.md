# IELTS Coach Frontend (Next.js 14)

Design vibe: Duolingo + Grammarly + Notion, using blue + white.

## Run locally
1) Install deps
   npm install

2) Start dev server
   npm run dev

Then open http://localhost:3000

## Routes
/onboarding
/diagnostic (and /diagnostic/listening, /reading, /writing, /speaking, /results)
/dashboard
/writing and /writing/feedback
/speaking and /speaking/feedback
/mock-tests (and /take, /results, /review)
/progress
/sign-in (placeholder)
/setup-mfa (placeholder)
/verify-mfa (placeholder)

## Notes
UI components are shadcn-style (button/card/badge etc) implemented locally for a ready-to-run template.
